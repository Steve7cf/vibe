- Node.js 18+

```bash

```


## Usage

1. **Add Music**: Click "Add Music" or drag & drop audio files
2. **Scan Folder**: Click "Scan" to import an entire music directory
3. **Play**: Double-click any track, or click the play button on album/artist cards
4. **Equalizer**: Click the EQ icon in the bottom bar
5. **Playlists**: Create playlists via the sidebar or Playlists view
6. **Settings**: Click the gear icon in the sidebar for all options
